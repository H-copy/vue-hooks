### Params

| 名称    | 说明                             | 默认值 |
| ------- | -------------------------------- | ------ |
| initDom | 初始Dom ref                      |        |



### Result

| 名称    | 说明             | 类型          |
| ------- | ---------------- | ------------- |
| state   | 当前scroll位置值 | { top, left } |
| element | 监听元素         | ref           |
